# Inventory Manager

Inventory Manager is a Python desktop application for laboratory inventory management. The project uses PyQt6 for the GUI and SQLite for persistent storage. This repository contains the application code, tests and supporting files, and detailed documentation in the `docs/` folder.

See the documentation for installation, usage, and development details: [Documentation index](docs/index.md)
