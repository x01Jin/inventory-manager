# Usage

Quick Start

1. Activate your virtual environment and run the application (see installation).
2. The database will be initialized automatically on first run.
3. The application launches a GUI (PyQt6) for interactive inventory management.

User Interface Overview

- **Dashboard**: Provides high-level inventory metrics and alerts.
- **Inventory**: Manage items, batches, stock movements and alerts.
- **Requisitions**: Create, modify, and process requisitions with full lifecycle support.
- **Requesters**: CRUD for personnel allowed to request inventory.
- **Reports**: Generate Excel-based reports for usage, stock, and expiration.
- **Settings**: Manage sizes, suppliers, brands and categories used across the app.

Common Workflows

- Add an Item: Use the Add Item dialog in Inventory, fill required fields, and save.
- Request Items: Create a requisition, select items, and confirm reservation.
- Process Returns: Use the Requisition preview to process returned or consumed items.

Shortcuts

- Double-click table rows to open editors.
- Search and filters exist on Inventory, Requisitions, and Requesters pages.
